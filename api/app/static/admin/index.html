자동 경로: -
전체 업로드는 새 데이터 적재 완료 후 이전 데이터를 자동 정리합니다.
업데이트 모드에서는 입력 데이터의 추가/수정/삭제 컬럼을 기준으로 변경분만 반영됩니다.
